Sireum Awas is an information flow analyzer and visualizer for
component-based systems. It consists of OSATE plugin, analysis
algorithms, and interactive model explorer.

The Awas OSATE plugin generates a self-contained web-page(here onwards
referred to as the Awas visualizer) that provides an interactive
environment to explore and analyze component-based systems. The
generated Awas visualizer is a self contained website. This allows the
user to explore the model even in devices that are not supported by
OSATE.

Awas is an `open source <https://github.com/sireum/v3-awas>`__ project released under
the `Simplified BSD <https://github.com/sireum/v3/blob/release/license.md>`__ license.
